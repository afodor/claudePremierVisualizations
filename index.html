About this project

This repository contains the simulation, statistical analysis, neural network modeling, and interactive 3D visualizations for PreMiEr. The simulation models 4,000 households with AR(1) microbiome dynamics, environmental sensors, and an AllergyANN neural network (R²=0.787). See the full manuscript (PDF) for details, or browse the source code on GitHub.
